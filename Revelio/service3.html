Usage Analysis

Visual Insight of Your Average Time Spent on Google Platforms

The interactive plots, generated using Google Takeout's My Activity Data, visualizes the time of usage calculated based on action timestamp log.

Overall Usage Time
Total activity time spent using Google Platforms by users, aggregated by four different time units: day, week, month, and year.
x-axis represents the time unit; y-axis represents the total time spent in certain time unit; Bar colors represent the different platform types.

Usage Time by Platform
Total activity time spent using each platform, aggregated by the selected time unit.
x-axis represents the time unit; y-axis represents the total time spent using the selected platform in certain time unit.

Weekly Usage Time Proportion by Platform
Weekly activity time for each activity, proportional to the total of weekly usage time.
x-axis of the figure represents starting date of calculation; y-axis represents the percentage of weekly usage time spent on the selected platform.
